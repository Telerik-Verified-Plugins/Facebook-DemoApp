Facebook Plugin Demo

The Facebook plugin allows your users to log in wihh their Facebook account. You can also interact with their account on their behalf after asking the relevant permissions.

Custom plugins don't work in the simulator and the AppBuilder companion app, so inside Telerik AppBuilder choose Run - Build - iOS / Android - App package.

Login

In case the user has the Facebook app installed, authentication is done by showing the in-app consent screen. If the app is not installed, a browser window is opened in order to show the online consent screen.
